# DS4021-Final-Project

## Software and Platform Selection

### Programming Language & Environment
- Python 3.11
- Jupyter Notebooks / JupyterLab
### Required Python Packages
- pandas – data cleaning and manipulation
- numpy – numerical operations
- scikit‑learn – preprocessing, model training, cross‑validation, evaluation
- matplotlib – plotting
- seaborn – statistical visualizations
- PyTorch – neural network modeling
- scipy – statistical utilities
### Operating System
- Compatible with macOS, Windows, and Linux
- Project developed mainly on macOS

## Documentation Map
Below is a list of the folders in the repository as well as their contents
- Outputs: Includes our EDA plots and other figures used in the presentation
- Data: Includes our training data(data used for our models to learn) and testing data(data used to test our best model on unseen data)
- Notebooks: Includes our notebooks for both EDA and our models as well as final test set
  - Models: Includes seperate notebooks for each of our four models(Penalized Linear Regression, Support Vector Regression, Gradient Boosted Decision Tree, Neural Network)
  - EDA/model: Includes our EDA notebook containing visualizations that introduce the structure as our data as well as our model notebook which contians our explanatory statistics
  - test_set_evalation: Notebook evaluating our best model on unseen testing data
- final_report.pdf: Our final report summarizes our motivations, methods, results, and conclusions
